# find_hitler_game
Учебный проект по Python/Flask

Игра, в которой пытаемся перейти по ссылкам от одной статьи Википедии до другой, а конкретно - "Гитлер, Адольф". Подробности в самой игре, при нажатии на кнопки "Справка"
